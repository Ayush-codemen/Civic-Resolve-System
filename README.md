# Civic-Resolve-System
Civic Resolve System is a smart web-based platform that enables citizens to report civic issues and track their resolution in real-time, improving transparency and efficiency in public services.

The system ensures transparency, accountability, and faster resolution through real-time tracking and smart complaint management.

---

## 🎯 Features

### 👤 User Module
- Register and Login
- Submit complaints with images and location
- Track complaint status in real-time
- View complaint history

### 🏛️ Admin Module
- Dashboard to manage complaints
- Assign issues to relevant departments
- Update complaint status (Pending → In Progress → Resolved)
- Monitor and analyze reports

### 📍 Smart Features
- Location-based complaint tagging
- Priority-based issue classification
- Notifications (Email/SMS)
- Data analytics and reporting

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript / React
- **Backend:** Node.js / Java (Spring Boot)
- **Database:** MongoDB / MySQL
- **APIs:** Google Maps API, Firebase (optional)

---

## 🔄 Workflow
1. User submits a complaint with details and location  
2. Complaint is stored in the database  
3. Admin reviews and assigns the issue  
4. Status is updated as the issue progresses  
5. User receives updates and tracks resolution
